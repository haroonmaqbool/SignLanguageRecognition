# Installation Instructions:
# =========================
# 1. Create a virtual environment:
#    python -m venv sign_lang_env
# 
# 2. Activate the virtual environment:
#    # On Windows:
#    sign_lang_env\Scripts\activate
#    # On macOS/Linux:
#    source sign_lang_env/bin/activate
#
# 3. Install dependencies:
#    pip install -r requirements.txt
#
# 4. Run the preprocessing:
#    python preprocessing.py
#
# 5. Train the models:
#    python train_model.py
#
# 6. Evaluate the models:
#    python evaluate_model.py
#
# 7. Run real-time detection:
#    python realtime_detection.py
#
# 8. Start the web application:
#    python app.py
#
# Notes:
# ======
# - gTTS (Google Text-to-Speech) is used for converting generated text to speech
# - Requires internet connection for gTTS to function
# - All other features work offline once models are trained
